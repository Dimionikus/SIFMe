SIFMe (Save It For Me) — is a program designed to assist Windows OS users.

Now in alpha version.

Current main features:

- File search by tags;
- Notification system.

Isues:

- currently no settings;
- currently only supports RUS localization;
- no backup function;
- data storage is not implemented in the intended way

SIFMe v0.1.0-alpha was created as a diploma project at a technical college.
